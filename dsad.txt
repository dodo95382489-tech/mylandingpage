집에가고싶다

1. 프로젝트 개요
서비스 소개

"집에가고싶다"는 누구나 한 번쯤 느끼는 "집에 가고 싶다"는 감정을 주제로 한 밈 기반 게임·커뮤니티 웹사이트입니다.

사용자는 집에 가고 싶다는 버튼을 누르며 공감에 참여하고, 집에 가고 싶은 이유를 작성하거나 다른 사람들과 공감할 수 있습니다. 또한 미로를 통과해 집으로 도착하는 게임을 즐길 수 있습니다.

2. 서비스 목표
사용자들의 공감대를 형성한다.
재미있는 밈 콘텐츠를 제공한다.
커뮤니티와 게임 요소를 결합한다.
SNS 공유를 통해 자연스러운 확산을 유도한다.

<script>
let count = 0;

const messages = [
"집은 당신을 기다리고 있습니다.",
"현재 집가고싶음 지수 상승 중.",
"퇴근까지 조금만 더.",
"오늘도 잘 버티고 계시는군요.",
"집이 가장 안전합니다.",
"당신의 집가고싶음이 기록되었습니다.",
"전국의 집순이, 집돌이가 응원합니다."
];

function goHome(){

    count++;

    document.getElementById("count").innerText = count;

    const random =
    messages[Math.floor(Math.random()*messages.length)];

    document.getElementById("message").innerText =
    random;
}

function addReason(){

    const input =
    document.getElementById("reason");

    const text =
    input.value.trim();

    if(text===""){
        alert("이유를 입력해주세요.");
        return;
    }

    const post =
    document.createElement("div");

    post.className = "post";

    post.innerText = text;

    document.getElementById("posts")
    .prepend(post);

    input.value = "";
}
</script>
