# 슬기로운 자격생활 (Plan)

## 1. 프로젝트 개요
- 프로젝트명: 슬기로운 자격생활
- 유형: 자격증 정보 + 시험 일정 + 학습 지원 포털
- 스타일: 네이버(Naver) 포털형 UI 기반 정보 서비스
- 목적: 포트폴리오용 웹사이트 제작

## 2. 타겟 사용자
- 모든 자격증 준비생
- 대학생 / 취준생 / 직장인 포함

## 3. 핵심 목표
- 자격증 정보를 한 곳에서 검색/확인
- 시험 일정 및 D-day 관리
- 사용자 개인 맞춤 자격증 관리

## 4. 주요 기능

### 4-1. 정보 기능 (50%)
- 자격증 검색 기능
- 자격증 상세 정보 제공
- 시험 일정 정보 제공
- 인기 자격증 리스트

### 4-2. 기능 기능 (50%)
- 시험 일정 캘린더
- D-day 계산 기능
- 즐겨찾기 (찜 기능)
- 추천 자격증 시스템
- 합격 후기/정보 제공

## 5. 페이지 구조

### 5-1. 메인 페이지
- 검색창 (핵심)
- D-day 시험 정보
- 인기 자격증
- 추천 자격증
- 커뮤니티/후기

### 5-2. 상세 페이지
- 자격증 정보
- 시험 일정
- 합격 기준
- 과목 정보

### 5-3. 캘린더 페이지
- 시험 일정 전체 보기
- 월별/주별 필터

### 5-4. 마이페이지
- 저장한 자격증
- D-day 관리
- 개인 학습 현황

### 5-5. 로그인/회원가입
- 클라이언트 기반 로그인 (localStorage)

## 6. 데이터 구조 (예시)
```json
{
  "name": "정보처리기사",
  "examDate": "2026-03-15",
  "category": "IT",
  "difficulty": "중",
  "tags": ["국가자격증", "취업"]
}
```

## 7. 기술 스택
- HTML
- CSS
- JavaScript (Vanilla)
- localStorage (로그인/찜 기능)
- JSON 데이터 관리

## 8. 로그인 구조
- 회원가입: 아이디/비밀번호 저장 (localStorage)
- 로그인 상태 유지
- 사용자별 찜 목록 저장

## 9. UI/UX 방향
- 네이버 스타일 포털 UI
- 정보 중심 레이아웃
- 빠른 검색 중심 구조
- 직관적인 카드 UI
